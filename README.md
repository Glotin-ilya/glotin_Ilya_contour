Контурное распознавание объектов (алгоритм Протасова)

Репозиторий содержит реализацию алгоритма распознавания объектов по их контуру, основанного на методе В. И. Протасова «Распознавание трёхмерных объектов на сложном фоне по части контура» (2001).

Алгоритм предназначен для распознавания объектов по полному или частичному контуру и инвариантен к повороту, масштабу и сдвигу.


Краткое описание алгоритма.
1. Загружается изображение.
2. оно преобразуется в черно-белое и бинаризуется.
3. Находятся внешние контуры обьектов.
5. Для него:
	-Удаляются дублирующие точки.
	-Рекурсивно выделяются характерные точки (углы) методом, похожим на алгоритм Протасова.
	-Рассчитать два признака:
		-Количество углов (numCorners)
		-Степень вытянутости (elongation = длинна / прямое расстояние)
6. На основании этих признаков принимается решение.


Требования

- С++
- Библиотека OpenCV


Сборка проекта 
cmake .. cmake --build .

Структура проекта
- main.cpp — основной исходный код
- папка images - примеры изображений


Запуск
./recognizer путь/к/изображению


Требования к изображению
- Обьект должен быь светлым на темном фоне.
- Лучше использовать черно-белые силуэты без шума.
- Чем четче контур, тем точнее распознавание.

Результаты
Алгоритм продемонстрировал инвариантность к масштабу и повороту: обьекты различных классов надежно различаются.
